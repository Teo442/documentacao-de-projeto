---

## Sobre o Sistema "AutoPark"

O **AutoPark** é um sistema projetado para gerenciar as operações de um estacionamento rotativo. Suas principais funcionalidades incluem:

* **Registro de Entrada:** Marcar a entrada de um veículo e associá-lo a uma vaga.
* **Registro de Saída:** Calcular o valor a ser pago com base no tempo de permanência e na tarifa.
* **Controle de Vagas:** Visualizar o status das vagas (livres/ocupadas) em tempo real.
* **Gerenciamento:** Permitir que administradores configurem tarifas e gerenciem usuários.

### Stack Tecnológica (Projetada)

A arquitetura do sistema foi planejada para utilizar as seguintes tecnologias:

* **Frontend:** Aplicação Web (SPA) em **React**.
* **Backend:** API REST em **Java** com **Spring Boot**.
* **Banco de Dados:** **MySQL**.
